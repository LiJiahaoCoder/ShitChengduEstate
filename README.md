# ShitChengduEstate

## 简介

近两年买了房，但房屋质量问题、以及开发商货不对板的问题维权无门；国内各种平台会限制发这类消息，所以通过这种方式来曝光无良开发商。

欢迎各位提PR曝光成都无良开发商和楼盘。

## PR 规范

目录结构：

```bash
|- A区
  |- 楼盘A
|- B区
  |- 楼盘B
|- ...
```

请按照上述格式 `行政区/楼盘名` 建立目录，并在下方目录添加引用方便直接跳转到对应楼盘曝光内容。
